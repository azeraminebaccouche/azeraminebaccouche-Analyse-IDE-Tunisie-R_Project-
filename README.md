# Analyse IDE Tunisie - R Project

R project with Quarto presentations and data analysis for Tunisia investment and employment analysis.

**Repository:** [https://github.com/azeraminebaccouche/Analyse-IDE-Tunisie-R_Project-](https://github.com/azeraminebaccouche/Analyse-IDE-Tunisie-R_Project-)

## Contents

- R scripts and Quarto presentations
- Data files (CSV, XLS)
- Generated HTML presentations
- Power BI files

## Files

- `r.R` - Main R script
- `rpresentation.qmd` - Quarto presentation source
- `R_revealjs.qmd` - Reveal.js presentation source
- Data files for Tunisia investment and employment analysis

## Requirements

- R
- Quarto
- Required R packages (install as needed)

## Usage

Open the project in RStudio or run the R scripts directly.
